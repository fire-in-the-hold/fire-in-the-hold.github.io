一個 class#

  def wallet (id)

  end
end

a16z = Fund.new
a16z.wallet 2134543 # 印出 wallet #2134543

initialize 與 繼承#

class Pool
  def order (id)
    puts "order ##{id}"
  end
end
class Fund < Pool
  def initialize(name)
    @name = name
    puts "new Fund #{@name}"
  end
  def wallet (id)
    puts "wallet ##{id}"
  end
end

a16z = Fund.new "a16z" # 印出 new Fund
a16z.order 2134543 # 印出 order #2134543

實體變數#

class Fund
  def initialize ( name )
    @name = name
  end
  
  def name # getter
    @name
  end

  def name=(new_name) # setter
    @name = new_name
  end
end

a16z = Fund.new('a16z')
puts a16z.name # 印出 a16z
puts a16z.name = "Andreessen Horowitz" # 印出 Andreessen Horowitz
puts a16z.name = "Andreessen Horowitz / a16z" # 印出 Andreessen Horowitz / a16z
puts a16z.name=("A16z ") # 印出 A16z

可以使用 attr_reader 與 attr_writer 簡化 getter 與 setter 寫法

class Fund
  attr_reader :name
  attr_writer :name
  def initialize ( name )
    @name = name
  end
end

a16z = Fund.new('a16z')
puts a16z.name # 印出 a16z
puts a16z.name = "Andreessen Horowitz" # 印出 Andreessen Horowitz
puts a16z.name = "Andreessen Horowitz / a16z" # 印出 Andreessen Horowitz / a16z
puts a16z.name=("A16z ") # 印出 A16z

類別變數#

存在於該類別中，比方你 new 多個 Fund 出來，你可以計算有多少個 Fund 被 new 出來

class Fund
  @@count = 0
  def initialize ()
    @@count += 1
  end
  
  def self.counter
    @@count
  end
end

5.times {
  Fund.new
}
p Fund.counter # 印出 5

開放類別#

ruby 特有的，會將兩個相同 class 進行融合，所以兩個呼叫都不會報錯

class Say
  def hello
    puts "hello"
  end
end
class Say
  def world
    puts "world"
  end
end

blog = Say.new
blog.hello # 印出 hello
blog.world # 印出 world

開放類別範例#

可以將原本的類別在融合一些功能上去

class Integer

存取控制#

預設都是 public 方法，所以 Pool Wallet 是公開方法 當看到 private 之後，都會為私有變數，所以 Order aaa bbb 都是私有方法

class Fund
  def pool
    puts 'Pool'
  end

  def wallet
    # 呼叫私有分法，請直接呼叫，不能使用 self.aaa 
    aaa
  end

  private
  def order
    puts 'Order'
  end

  def aaa
    puts 'Hello'
  end

  def bbb
  end
end

a16z = Fund.new

# 公開方法可以呼叫
a16z.pool

# 私有方法不可以呼叫 !
a16z.order

# 可以透過 send + 符號 偷偷呼叫 私有方法
a16z.send(:order)

# 會印出 Hello
a16z.wallet

存取控制2#

protected 他介於 public 與 private 中間，同一個類別內或是同一個 package，或是繼承它的子類別可以自由取用，但如果不是的話則不可存取。

class Fund
  def pool
    puts order
    puts self.order
  end

  protected
  def order
    puts 'Order'
  end
end

a16z = Fund.new

# 公開方法可以呼叫，且會印出兩個 Order
a16z.pool

# 無法呼叫
a16z.order
