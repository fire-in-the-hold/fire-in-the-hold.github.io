學習 Ruby 123 - 11 rake

June 13, 2022

類似 makefile 檔案，用 ruby 寫的

建立一個 rakefile 檔案

rakefile

desc "這是一段描述"
task :send do
  puts 'send email'
  sleep 2
  puts 'send discord'
  sleep 2
  puts 'done'
end

task :get_amount do
  puts "amount $10000"
end

desc "取得錢包"
task :wallet => :get_amount do
  puts 'done'
end

namespace :fund do
  desc "從池子取得錢包"
  task :wallet => :get_amount do
    puts 'done'
  end
end

可以執行 rake 指令

$ cd ~/ruby_project

$ ls
./ rakefile

$ rake -T
rake fund:wallet  # 從池子取得錢包
rake send # "這是一段描述"
rake wallet  # 取得錢包

$ rake send
send email
send discord
done

$ rake wallet
amount $10000
done

$ rake fund:wallet
amount $10000
done
