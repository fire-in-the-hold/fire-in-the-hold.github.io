學習 ROR 123 - 1 建立專案

June 14, 2022

Ruby on Rails#

他是一個透過 Ruby 建立的 Web server，使用 ROR 前，請先裝上 nodejs

# 安裝指定版本 ROR
$ gem install rails -v 6.1.6

# 建立一個新專案
$ rails new hello-project

# 建立一個指定版本 ROR 專案
$ rails _6.1.6_ new hello-project

# 可跳過，安裝套件
$ bundle install

# 可跳過，可能要先安裝 nodejs 然後 yarn 過安裝套件
$ rails webpacker:install

# 啟動伺服器
$ rails server
$ rails s # 這是縮寫

rvm 操作#

# 安裝 rvm
$ curl -L get.rvm.io | bash -s stable

# rvm 安裝 ruby
$ rvm install "ruby-2.6.8"

# rvm 版本查看
$ rvm list known

使用 mongodb#

gemfile 新增 gem "mongoid"

# gem install 
$ bundle install

# 建立連線資料 檔案在 config/mongoid.yml
$ rails g mongoid:config
