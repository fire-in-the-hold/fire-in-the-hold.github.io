如何建置 Hugo 部落格

June 5, 2022

這是一個使用 Hugo 快速建立的部落格，他能協助你快速的建立一個部落格網站，並且可將 Code 放置於 Github.io 免費的伺服器空間。這種網站好處就是，免伺服器費用，可以高度自定化，對於 SEO 而言也非常友善。

要先知道的東西#

  1. Hugo 本次的主角，一系列工具可以讓你快速產生程式碼。

  2. Markdown 是一個可以透過文字的方式，做出各種排版的一種語法格式。

  3. Git 版本控制工具，讓你可以對於文字作版本控制。

  4. Version control 就是你文字編輯過程都有一個紀錄，只要處理一個檔案即可。可以隨時切回以前的版本，而不用把桌面放的滿滿的備份檔案。

  5. Github 存放 Version control 所有歷史資料的服務，另外也提供一些免費靜態網站空間，讓你可以存放 HTML。

  6. Homebrew Macos 套件管理的套件管理

  7. VSCode 一個功能特別多的文字編輯器

環境建置#

本篇以 Macos 開始

  1. 打開 Terminal 輸入以下指令安裝 Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. 輸入以下指令安裝 Hugo
brew install hugo
  1. 輸入指令建立一個 Hugo 檔案資料夾，裡面會包含一些 Hugo 的程式碼
hugo new site ${你的資料夾名稱}
  1. 到 https://themes.gohugo.io/ 挑選一個喜歡的模板下載，下載後解壓縮，將整個資料夾，放到上一步所產生資料夾內的 ${你的資料夾名稱}/themes/
  • archetypes/: 文章預設配置
  • content/: 放文章的地方
  • data/: (空的)
  • layout/: 網站一些的配置，包含頁首頁尾那些
  • static/: 放圖片、CSS等檔案的地方
  • themes/: 你使用的模板配置 <- 將模板放置於此
  • config.toml: Hugo網站的核心配置檔
  • public/: Hugo 產生的網頁檔案，可以上傳至 Github 就變成你的部落格
  1. 變更 config.toml 設定，裡面加上以下設定，套上模板，定義你的網頁路徑，另外上面也有一些資訊你可以跟著修改，比方部落格名稱等
theme = "${你模板名稱}"

[menu]
  [[menu.main]]
    identifier = "post"
    url = "/post/"
    # title will be shown when you hover on this menu link
    title = ""
    weight = 1

Github 操作#

  1. 建立一個 Blog Repo 設定為公開，名稱為 ${github account}.github.com

  2. 建立一個 Blog Core Repo 存放你的 Hugo 程式碼

  3. 將 Hugo 這個資料夾丟上 Blog Core，並且建立 .gitignore 裡面寫著 /public/ 代表這個檔案裡下的 public 資料夾不要上傳，另外 git clone Blog Repo 到這個資料夾底下取名為 public

如何操作#

  1. 建立一篇貼文

  2. 使用 hugo server 在自己電腦上查看

  3. 使用 hugo 產生網頁檔案

  4. 上傳 public 的檔案到 Github 上即可

一些指令#

啟動 Hugo 伺服器可以快速預覽

hugo serve

創建一篇部落格貼文

hugo new content/posts/2021/${English post name}.md
